## Hack on the code

### Prerequisites

* [Node](http://nodejs.org/)
* [Ruby](https://www.ruby-lang.org/en/) - r参考
* [Sass](http://sass-lang.com/) `gem install sass` (>=3.3 引入版本)

示例：

```sh
npm install
```

### Running

```sh
npm start
```

运行服务
[http://localhost:3000/service-Work-demo/](http://localhost:3000/service-Work-demo/
